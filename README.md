# Bson To Json

IO
```bash
USAGE: bson-to-json io [--use-std-out] [--overwrite] [--input <input>] [--output <output>]

OPTIONS:
  --use-std-out           Use standard out instead of creating a new file.
  -f, --overwrite         Automatically overwrite existing files
  -i, --input <input>     Path to file to convert
  -o, --output <output>   Destination to write output
  --version               Show the version.
  -h, --help              Show help information.

```

Batch (Default)
```

USAGE: bson-to-json batch [--overwrite] <input-files> ...

ARGUMENTS:
  <input-files>           Files to convert

OPTIONS:
  -f, --overwrite         Automatically overwrite existing files
  --version               Show the version.
  -h, --help              Show help information.
```
